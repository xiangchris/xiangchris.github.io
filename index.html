Rotisserie Game Engine

A multiplatform 2D scriptable game engine written in C++ which features Lua scripting support. Developed over a 9 week cycle. The engine uses SDL for game rendering, Box2D for 2D physics, LuaBridge for scripting, and Dear ImGui for editor interface rendering. Used Visual Studio as the primary IDE and imported engine to XCode for macOS support.
